# Screenshots

This directory contains screenshots for the release and documentation.

## Screenshots to Add

1. **Editor Screen** - Screenshot showing the main editor UI
2. **Repository List** - Screenshot with repositories
3. **Preview Pane** - Screenshot with markdown preview
4. **Settings Screen** - Settings UI screenshot
5. **Pull Request Screen** - PR interface screenshot

## Format

Add screenshots to this directory before release. Screenshots:
- Format: PNG
- Orientation: Portrait preferred
- Resolution: High resolution (3:4)
- Purpose: Release notes, Google Play Store listing

## Commands

```bash
# Add screenshot
cp /path/to/screen.png /sandbox/.openclaw/workspace/SentinelEditor/screenshots/

# Clean up
rm -f screenshots/screenshots-to-remove.png
```
