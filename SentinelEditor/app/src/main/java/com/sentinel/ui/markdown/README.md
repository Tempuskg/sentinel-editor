# Markdown UI Components

This package provides Compose components for the Markdown editor.

## Overview

The editor implements a WYSIWYG markdown editing experience with:

- Split-pane layout (editor/preview)
- Syntax highlighting
- Inline code highlighting
- Link preview
- Table support
- Image rendering
- Syntax highlighting theme

## Components

### EditorPane
- Input field for markdown content
- Cursor tracking
- Scroll to cursor support

### MarkwonRenderer
- Renders markdown to HTML
- Provides syntax highlighting
- Handles inline elements
- Provides preview

### SyntaxHighlighter
- Applies syntax highlighting
- Handles code blocks
- Provides theme colors
