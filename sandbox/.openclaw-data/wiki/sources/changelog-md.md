# Source: CHANGELOG v1.0.0

Initial release changelog for Sentinel Editor.

## Key Takeaways

- **Version:** 1.0.0 (2026-04-11)
- Core features: WYSIWYG editor, GitHub OAuth 2.0, Room database, PR management, dark theme
- Editor: cursor tracking, sync scroll, syntax highlighting, link/table/image support
- Persistence: Room offline editing, cursor/scroll persistence, theme prefs, sync tracking
- GitHub: repo listing, file browsing, content fetching, commits, PRs, branches, token refresh with PKCE, rate limiting
- Security: token encryption with Android KeyStore, PKCE OAuth, rate limiting, privacy policy
