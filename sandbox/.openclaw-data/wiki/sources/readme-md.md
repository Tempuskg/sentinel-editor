# Source: Sentinel Editor README

The main README for the Sentinel Editor project, documenting features, architecture, and usage.

## Key Takeaways

- Professional WYSIWYG markdown editor for Android with GitHub integration
- Built with Jetpack Compose (BOM 2024.12.01), Room 2.6.1, Retrofit 2.11.0
- Custom editor implementation — no third-party editor library
- Features: WYSIWYG editing, GitHub OAuth, offline editing, PR management, syntax highlighting, diff view
- Architecture covers: token auth flow, repository/file listing with caching, custom markdown editor, Room persistence, commit workflow
- 6 screens defined in architecture documents
- All dependencies Apache-2.0/MIT/BSD compliant
